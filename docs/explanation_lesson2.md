## 1차시 핵심 3문장

1. **동결(Freeze-out):** 반응률 Γ이 팽창률 H보다 느려지면 (Γ/H < 1) 평형이 깨진다
2. **중성자 붕괴:** 중성자는 시간이 지나면 일정 비율로 양성자로 변한다 (τ ≈ 879초)
3. **헬륨 질량비:** $Y_p = 2X_n$ (중성자가 제한 반응물)

그런데 **왜** 이 온도인가? 바로 이 질문에 답하기 위해 2차시에서 그 이유를 알아본다.

# 제1장. 가역 반응이란?

## 1.1 화학에서의 가역 반응

먼저 중학교 과학에서 배운 가역 반응을 떠올려 보자.

### 예시 1: 물의 상태 변화

$$\text{물(액체)} \rightleftarrows \text{물(기체)}$$

| 방향 | 이름 | 조건 |
|------|------|------|
| → | 증발 | 열을 받으면 |
| ← | 응결 | 열을 잃으면 |

### 예시 2: 탄산음료

$$\text{CO}_2\text{(기체)} + \text{H}_2\text{O} \rightleftarrows \text{H}_2\text{CO}_3$$

- 뚜껑을 열면: ← 방향 (기체가 나옴)
- 뚜껑을 닫으면: → 방향 (녹아 있음)

## 1.2 평형의 진짜 의미 ★★★

> **평형 = 반응이 멈춘 것이 아니다!**

이 점을 명확히 이해해야 한다. 평형 상태에서도 양방향 반응이 **동시에** 일어난다.
그러나 **정반응 속도 = 역반응 속도**이기 때문에 알짜 변화가 없어 보일 뿐이다.

```
정반응 →  🔄  ← 역반응
    (같은 속도로 진행)
```

**비유:** 예를 들어 에스컬레이터에서 반대로 걷는 상황을 생각해 보자:
- 위로 올라가는 속도 = 내가 걸어 내려가는 속도
- 따라서 제자리에 있는 것처럼 보이지만, 둘 다 움직이고 있다!

# 제2장. 중수소 생성과 광분해

## 2.1 핵반응도 가역이다!

$$p + n \rightleftarrows D + \gamma$$

| 방향 | 반응 | 이름 | 결과 |
|------|------|------|------|
| → | $p + n \to D + \gamma$ | **중수소 생성** | D가 만들어짐 |
| ← | $D + \gamma \to p + n$ | **광분해(Photodissociation)** | D가 깨짐 |

## 2.2 광분해란?

**광(Photo)** + **분해(Dissociation)** = 빛(광자)에 의해 분해됨

> 🔴 **블랙박스:** 중수소 결합에너지 ~2.2 MeV는 핵물리 실험에서 측정된 값이다.

- 중수소(D)의 결합에너지: ~2.2 MeV
- 따라서 광자(γ)의 에너지가 2.2 MeV 이상이면 → D를 깰 수 있다

## 2.3 망치 비유 ★★★

광자를 **망치**로 비유해보자:

| 온도 | 광자(망치)의 상태 | 결과 |
|------|-----------------|------|
| **고온** | 에너지 높은 망치가 많다 | D를 계속 부순다 |
| **저온** | 에너지 낮은 망치만 남는다 | D가 살아남는다 |

```
고온: 🔨🔨🔨🔨🔨 → D 즉시 파괴!
      (강한 망치 많음)

저온: 🔨 → D 살아남음!
      (약한 망치만)
```

## 2.4 왜 고온에서 D가 쌓이지 않는가?

그렇다면 고온에서는 어떤 일이 일어나는가?
1. 물론 D가 만들어지기도 한다 (정반응)
2. 그러나 **바로 깨진다** (역반응이 훨씬 빠름)
3. 그 결과: D의 알짜 축적량 ≈ 0

> **핵심:** 사실 "D가 안 만들어진다"가 아니라 "만들어져도 바로 깨진다"는 점이 중요하다! 이 차이가 물리적으로 큰 의미를 갖는다.

# 제3장. 중수소 병목 (Deuterium Bottleneck)

## 3.1 병목이란?

**병목(Bottleneck)** = 좁은 구간 때문에 전체 흐름이 막히는 현상

일상에서도 비슷한 예시를 찾을 수 있다:
- 예를 들어 고속도로 톨게이트
- 또한 병의 좁은 목 부분
- 그리고 지하철 환승 통로

## 3.2 BBN에서의 병목 ★★★

그렇다면 BBN에서 병목은 어디에 있는가? 핵합성 반응 순서를 보자:
```
p + n → D → ... → He-4
        ↑
     [여기가 병목!]
```

다시 말해:
1. **고온:** D가 쌓이지 않음 → 따라서 다음 단계로 못 감
2. **저온:** D가 쌓이기 시작 → 비로소 핵합성이 진행!

> **중수소 병목 = 핵합성의 "시작 버튼"**

## 3.3 병목이 열리는 순간

온도가 내려가면 다음과 같은 일이 일어난다:
1. 먼저 광자의 에너지가 낮아진다
2. 그 결과 광분해 속도가 느려진다
3. 따라서 정반응 > 역반응이 된다
4. 마침내 D가 **처음으로 살아남기 시작**한다!

바로 이 순간을 **"병목이 열린다"**고 표현한다.

# 제4장. 스위치 함수

## 4.1 토이 모델에서의 단순화

정밀한 핵반응 네트워크를 풀지 않고도 간단한 **스위치 함수**로 근사할 수 있다.

> 🔴 **블랙박스:** 스위치 함수는 복잡한 Saha 방정식의 결과를 단순화한 것이다.

### 계단 함수 (가장 단순)

$$B(T) = \begin{cases} 0 & (T > T_D) \\ 1 & (T \le T_D) \end{cases}$$

- $T > T_D$이면: 스위치 OFF (핵합성 불가)
- 반면 $T \le T_D$이면: 스위치 ON (핵합성 시작)

```
B(T)
  1 |        ________
    |       |
  0 |_______|
    +-----------------→ T
           T_D
    (온도 감소 방향 →)
```

### 시그모이드 함수 (더 현실적)

한편, 더 현실적인 모델을 원한다면 시그모이드 함수를 사용할 수 있다:

$$B(T) = \frac{1}{1 + \exp\left(\frac{T - T_D}{\Delta}\right)}$$

- $\Delta$가 작으면: 급격한 전환
- 반대로 $\Delta$가 크면: 부드러운 전환

```
B(T)
  1 |        _____
    |      _/
    |    _/
  0 |___/
    +-----------------→ T
           T_D
```

## 4.2 스위치의 의미

| B(T) 값 | 의미 |
|---------|------|
| 0 | 핵합성 꺼짐 (D 살아남지 못함) |
| 0.5 | 전환점 (T = T_D) |
| 1 | 핵합성 켜짐 (D 쌓이기 시작) |

## 4.3 스위치를 쓰는 장점과 위험

| 장점 | 위험 |
|------|------|
| 복잡한 계산을 **압축** | 물리적 디테일을 **생략** |
| 개념적으로 명확 | "왜 이 온도인가?"를 숨김 |
| 수치적으로 안정 | 과도하게 단순화할 수 있음 |

> **정당한 사용:** 스위치가 무엇을 대신하는지(광분해 경쟁) 이해할 때

# 제5장. 두 가지 모형 비교

## 5.1 모형 A: 고정 $T_{\text{nuc}}$ (1차시)

- $T_{\text{nuc}} = 0.07$ MeV를 **주어진 값**으로 사용
- 장점: 단순하고 빠른 계산이 가능
- 그러나 단점: "왜 이 값인가?"에 대한 설명이 없음

## 5.2 모형 B: 자동 $T_{\text{nuc}}$ (2차시)

- 스위치 함수에서 B(T) = 0.5가 되는 온도를 찾음
- 장점: 물리적 의미가 명확함
- 반면 단점: 스위치 함수 자체가 근사임

## 5.3 두 모형의 차이

일반적으로:
- 두 모형의 결과는 **비슷**하다 (같은 T_D를 사용하면)
- 그러나 모형 B가 **확장성**이 좋다 (T_D를 바꾸면 결과를 예측할 수 있음)
- 따라서 물리적 이해를 위해서는 모형 B가 더 적절하다

## 5.4 병목 지연의 효과

**만약 병목이 늦게 열리면?** (즉, T_D가 더 낮으면)

1. 먼저 핵합성 시작이 늦어진다
2. 그 사이 중성자가 더 오래 붕괴한다
3. 따라서 $X_n$이 더 낮아진다
4. 결국 $Y_p = 2X_n$도 **감소**한다

> **핵심:** 다시 말해, 병목 타이밍이 최종 헬륨 비율을 결정한다!

# 제6장. 세 가지 핵심 질문과 답 ★★★ 필수

## 질문 1: 왜 온도가 충분히 내려가도 중수소(D)가 바로 쌓이지 않는가?

**답:**
고온에서는 **광분해(역반응)**가 우세하기 때문이다.
다시 말해, 중수소가 만들어지더라도 고에너지 광자에 의해 바로 깨진다.
따라서 온도가 충분히 내려가야 광분해가 약해지고, 그제서야 중수소가 "살아남기" 시작한다.

## 질문 2: "가역 반응" 관점에서 $p+n \rightleftarrows D+\gamma$를 어떻게 이해할 수 있는가?

**답:**
이 반응은 **정반응(생성)**과 **역반응(광분해)**이 동시에 일어나는 가역 반응이다.

- 고온: 역반응(광분해)이 우세 → D가 쌓이지 않음
- 저온: 정반응(생성)이 우세 → D가 쌓이기 시작

평형 상태에서는 두 반응이 같은 속도로 일어나지만, 온도가 내려가면 평형이 깨지고 정반응 방향으로 이동한다.

## 질문 3: '핵합성 시작 온도' $T_{\text{nuc}}$는 무엇을 의미하며, 1차시의 가정과 어떤 관계가 있는가?

**답:**
$T_{\text{nuc}}$는 **"중수소 병목이 열리는 온도"**다.
이 온도 이하에서 중수소가 살아남기 시작하고, 이후 핵합성이 진행된다.

1차시에서는 이를 고정값(0.07 MeV)으로 **가정**했지만,
실제로는 광분해와 생성의 경쟁에 의해 **물리적으로 결정**된다.
스위치 함수는 이 복잡한 경쟁을 단순화한 것이다.

# 제7장. 오해하기 쉬운 개념

## 오해 1: "병목 = D가 아예 안 만들어짐"

**정정:** D는 고온에서도 만들어진다! 다만 **바로 깨질 뿐**이다.

## 오해 2: "온도만 낮아지면 즉시 결합한다"

**정정:** 정반응과 역반응이 **경쟁**한다. 어느 쪽이 우세한지가 중요하다.

## 오해 3: "스위치는 임의 설정이다"

**정정:** 스위치는 복잡한 물리(광분해 경쟁)를 **압축**한 것이다. 압축의 이유가 명확해야 정당하다.

# 부록 A. 수식 보충 (선택 읽기)

## A.1 시그모이드 함수의 유래

시그모이드 함수는 로지스틱 함수에서 왔다:

$$\sigma(x) = \frac{1}{1 + e^{-x}}$$

온도 의존성을 넣으면:

$$B(T) = \frac{1}{1 + \exp\left(\frac{T - T_D}{\Delta}\right)}$$

- $T \gg T_D$: 지수 부분이 크다 → B ≈ 0
- $T = T_D$: 지수 = 1 → B = 0.5
- $T \ll T_D$: 지수 부분이 작다 → B ≈ 1

## A.2 병목 온도의 물리적 유도 (심화)

실제로 병목 온도는 **사하 방정식(Saha equation)**에서 유도된다:

$$\frac{n_D}{n_p n_n} = \left(\frac{m_D T}{2\pi}\right)^{-3/2} \exp\left(\frac{B_D}{T}\right)$$

여기서 $B_D \approx 2.2$ MeV는 중수소 결합에너지다.

광자-바리온 비가 매우 크기 때문에 (~10^9), 실제 병목 온도는 결합에너지보다 훨씬 낮다. 즉, 광자가 워낙 많아서 결합에너지 이하에서도 D를 깰 수 있는 것이다.
