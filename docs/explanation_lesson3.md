# 들어가기 전에: 1-2차시 복습

## 1차시 핵심 3문장

1. **동결(Freeze-out):** 반응률 Γ이 팽창률 H보다 느려지면 (Γ/H < 1) 평형이 깨진다
2. **중성자 붕괴:** 중성자는 시간이 지나면 일정 비율로 양성자로 변한다 (τ ≈ 879초)
3. **헬륨 질량비:** $Y_p = 2X_n$ (중성자가 제한 반응물)

## 2차시 핵심 3문장

1. **가역 반응:** $p+n \rightleftarrows D+\gamma$에서 정반응과 역반응이 경쟁한다
2. **중수소 병목:** 고온에서는 광분해(역반응)가 우세하여 D가 쌓이지 않는다
3. **핵합성 시작:** 온도가 내려가 병목이 열리면 ($T < T_{nuc}$) 핵합성이 시작된다

## 3차시에서 배울 것

그렇다면 3차시에서는 무엇을 배울까? 바로 **"어떻게 He-4까지 가는가"**를 다룬다!

# 제1장. 핵종의 표기법

## 1.1 핵종이란?

**핵종(nuclide)** = 특정한 양성자 수(Z)와 중성자 수(N)를 가진 원자핵

그런데 같은 원소라도 중성자 수가 다르면 다른 핵종이다. 이를 동위원소라고 한다.

## 1.2 두 가지 숫자: A와 Z

| 기호 | 이름 | 의미 |
|------|------|------|
| **A** | 질량수 (mass number) | 핵자(양성자+중성자) 총 개수 |
| **Z** | 원자번호 (atomic number) | 양성자 개수 |

> **관계:** $A = Z + N$ (N = 중성자 개수)

## 1.3 표기법

$${}^A_Z X$$ 또는 간단히 ${}^A X$

예시:
- 양성자: ${}^1_1 p$ 또는 $p$ (A=1, Z=1)
- 중성자: ${}^1_0 n$ 또는 $n$ (A=1, Z=0)
- 중수소: ${}^2_1 D$ 또는 $D$ (A=2, Z=1)
- 삼중수소: ${}^3_1 T$ 또는 $T$ (A=3, Z=1)
- 헬륨-3: ${}^3_2\text{He}$ 또는 $\text{He-3}$ (A=3, Z=2)
- 헬륨-4: ${}^4_2\text{He}$ 또는 $\text{He-4}$ (A=4, Z=2)

## 1.4 이번 차시에서 다루는 6개 핵종

| 핵종 | 이름 | A | Z | N |
|------|------|---|---|---|
| $n$ | 중성자 | 1 | 0 | 1 |
| $p$ | 양성자 | 1 | 1 | 0 |
| $D$ | 중수소 | 2 | 1 | 1 |
| $T$ | 삼중수소 | 3 | 1 | 2 |
| ${}^3\text{He}$ | 헬륨-3 | 3 | 2 | 1 |
| ${}^4\text{He}$ | 헬륨-4 | 4 | 2 | 2 |

# 제2장. 보존법칙 ★★★

## 2.1 화학에서의 보존법칙 복습

먼저 화학 반응에서의 보존법칙을 복습하자. 화학 반응에서 원자의 종류와 개수는 보존된다:

$$2\text{H}_2 + \text{O}_2 \to 2\text{H}_2\text{O}$$

- 왼쪽: H 4개, O 2개
- 오른쪽: H 4개, O 2개 ✓

## 2.2 핵반응에서의 보존법칙

마찬가지로 핵반응에서도 보존되는 것이 있다:

1. **질량수(A) 보존:** 핵자(양성자+중성자)의 총 개수는 변하지 않는다
2. **전하(Z) 보존:** 또한 양성자의 총 개수도 변하지 않는다

## 2.3 예시: 중수소 생성 반응

$$p + n \to D + \gamma$$

**보존 검사:**
| | 왼쪽 | 오른쪽 |
|--|------|--------|
| A | 1 + 1 = 2 | 2 + 0 = 2 ✓ |
| Z | 1 + 0 = 1 | 1 + 0 = 1 ✓ |

> **참고:** 광자($\gamma$)는 A=0, Z=0이다.

## 2.4 예시: He-4 생성 반응

$$T + D \to {}^4\text{He} + n$$

**보존 검사:**
| | 왼쪽 | 오른쪽 |
|--|------|--------|
| A | 3 + 2 = 5 | 4 + 1 = 5 ✓ |
| Z | 1 + 1 = 2 | 2 + 0 = 2 ✓ |

## 2.5 보존법칙의 중요성

보존법칙은 왜 중요한가?
- 먼저 반응식이 **타당한지 검증**할 수 있다
- 또한 보존이 깨지면 그 반응은 **불가능**하다
- 그러나 주의! 보존을 만족해도 **반드시 많이 일어나는 것은 아니다** (반응률 문제)

# 제3장. 8개 핵심 반응

## 3.1 반응 표기법

핵물리에서는 반응을 다음과 같이 표기한다:

$$A(b,c)D$$

이 표기법의 의미는: A에 b가 들어가서 c가 나오고 D가 남는다는 것이다.
다시 말해: $A + b \to c + D$

## 3.2 8개 반응 목록

### 반응 1: 중수소 생성
$$p(n,\gamma)D \quad \Leftrightarrow \quad p + n \to D + \gamma$$

### 반응 2: He-3 생성 (p 경로)
$$D(p,\gamma){}^3\text{He} \quad \Leftrightarrow \quad D + p \to {}^3\text{He} + \gamma$$

### 반응 3: T 생성 (n 경로)
$$D(n,\gamma)T \quad \Leftrightarrow \quad D + n \to T + \gamma$$

### 반응 4: He-3 생성 (D+D)
$$D(D,n){}^3\text{He} \quad \Leftrightarrow \quad D + D \to {}^3\text{He} + n$$

### 반응 5: T 생성 (D+D)
$$D(D,p)T \quad \Leftrightarrow \quad D + D \to T + p$$

### 반응 6: He-4 생성 (T 경로)
$$T(D,n){}^4\text{He} \quad \Leftrightarrow \quad T + D \to {}^4\text{He} + n$$

### 반응 7: He-4 생성 (He-3 경로)
$${}^3\text{He}(D,p){}^4\text{He} \quad \Leftrightarrow \quad {}^3\text{He} + D \to {}^4\text{He} + p$$

### 반응 8: 교환 반응
$${}^3\text{He}(n,p)T \quad \Leftrightarrow \quad {}^3\text{He} + n \to T + p$$

## 3.3 각 반응의 보존 검사

| 반응 | A 보존 | Z 보존 |
|------|--------|--------|
| 1. p+n→D+γ | 1+1=2+0 ✓ | 1+0=1+0 ✓ |
| 2. D+p→He3+γ | 2+1=3+0 ✓ | 1+1=2+0 ✓ |
| 3. D+n→T+γ | 2+1=3+0 ✓ | 1+0=1+0 ✓ |
| 4. D+D→He3+n | 2+2=3+1 ✓ | 1+1=2+0 ✓ |
| 5. D+D→T+p | 2+2=3+1 ✓ | 1+1=1+1 ✓ |
| 6. T+D→He4+n | 3+2=4+1 ✓ | 1+1=2+0 ✓ |
| 7. He3+D→He4+p | 3+2=4+1 ✓ | 2+1=2+1 ✓ |
| 8. He3+n→T+p | 3+1=3+1 ✓ | 2+0=1+1 ✓ |

# 제4장. 반응 네트워크 지도 ★★★

## 4.1 네트워크란?

**네트워크** = 여러 요소(노드)가 연결(엣지)로 이어진 구조

이 경우:
- **노드:** 핵종 (n, p, D, T, He-3, He-4)
- **엣지:** 반응 (화살표)를 의미한다

## 4.2 BBN 반응 네트워크 지도

```
        n ←─────────────────────────┐
        │                           │
        ▼ (1)                      (8)
   p ──→ D ──────────────────────→ He-4
        │ ╲                      ╱ ↑
        │  ╲(2)            (7)╱   │
        │   ╲                ╱     │
       (3)   ╲──→ He-3 ──╱       (6)
        │         │               │
        ▼         │(8)            │
        T ←───────┘               │
        │                         │
        └─────────────────────────┘
```

## 4.3 D가 "문"인 이유 ★★★

네트워크를 자세히 보면 흥미로운 패턴이 보인다:

1. **모든 경로가 D를 통과한다!**
   - 먼저 n + p → D (중수소 생성)
   - 그 다음 D → T 또는 He-3 → He-4

2. **병목과의 연결**
   - 2차시에서 배운 병목 = D가 쌓이기 시작하는 시점
   - 따라서 D가 쌓이지 않으면 → 다음 단계로 못 감

> **결론:** 다시 말해, D는 핵합성의 "문(gate)"이다!

## 4.4 분기점 이해

그렇다면 D에서 출발하면 어디로 갈 수 있는가? 두 경로가 있다:

**경로 1: T를 거쳐 He-4로**
```
D → T → He-4
```

**경로 2: He-3을 거쳐 He-4로**
```
D → He-3 → He-4
```

흥미롭게도 두 경로 모두 결국 **He-4로 합류**한다! 이것이 바로 He-4가 많이 생성되는 구조적 이유이다.

# 제5장. 왜 He-4가 많이 생성되는가? ★★★

## 5.1 세 가지 이유

그렇다면 He-4가 BBN의 주요 생성물인 이유는 무엇인가? 세 가지 이유가 있다:

### 이유 1: 안정성 (결합에너지)

> 🔴 **블랙박스:** 결합에너지 값들은 핵물리 실험에서 측정된 값이다.

| 핵종 | 핵자당 결합에너지 |
|------|------------------|
| D | 1.1 MeV |
| T | 2.8 MeV |
| He-3 | 2.6 MeV |
| **He-4** | **7.1 MeV** ← 가장 안정! |

He-4는 핵자당 결합에너지가 가장 높다. 따라서 **가장 안정**하다.
그리고 안정한 곳에 도달하면 더 이상 변하지 않는다!

### 이유 2: 네트워크 구조

- 여러 경로가 모두 **He-4로 합류**한다
- 예를 들어 D → T → He-4
- 또한 D → He-3 → He-4
- 결국 흐름이 한 곳으로 모이면 그곳에 쌓인다!

### 이유 3: 제한 반응물 (중성자)

- 중성자는 자유 상태에서 **붕괴**한다 (τ ≈ 879초)
- 그리고 He-4 하나를 만들려면 **중성자 2개**가 필요하다
- 따라서 중성자가 제한 반응물이므로 → 모든 중성자가 He-4로 간다!

## 5.2 중간물질의 행동

그런데 D, T, He-3는 왜 최종 생성물이 아닌가?

- 왜냐하면 만들어지자마자 **다음 반응으로 소비**되기 때문이다
- 그 결과 그래프에서 **"올라갔다 내려가는"** 패턴을 보인다
- 즉, 이들은 **중간물질(intermediate)**이며, 최종 생성물이 아니다

```
풍부도
↑     D, T, He-3
│      ╱╲
│     ╱  ╲    He-4
│    ╱    ╲  ╱────
│   ╱      ╲╱
│  ╱
│ ╱
└─────────────────→ 시간
```

# 제6장. 민감도와 반응률

## 6.1 반응률이란?

**반응률** = 단위 시간당 반응이 일어나는 횟수

반응률이 높으면:
- 먼저 그 반응이 빠르게 진행된다
- 또한 네트워크에서 그 경로가 "강한" 경로가 된다

## 6.2 민감도 실험

그렇다면 특정 반응률을 바꾸면 결과가 어떻게 변하는가?

**예시:** 반응 6 (T+D→He4+n)의 반응률을 2배로 하면?

- 먼저 He-4 생성이 빨라진다
- 그 결과 T가 더 빨리 소비된다
- 최종 He-4 양은? (시뮬레이션으로 확인!)

## 6.3 주요 경로 찾기

민감도 실험으로 **어느 반응이 중요한지** 알 수 있다:

- 반응률을 바꿨을 때 결과가 크게 변하면 → **중요한 반응**
- 반면 반응률을 바꿔도 결과가 비슷하면 → 상대적으로 덜 중요하다

# 제7장. 세 가지 핵심 질문과 답 ★★★ 필수

## 질문 1: 병목이 풀린 뒤 He-4가 "빠르게" 만들어지는 이유는 무엇인가?

**답:**
병목이 열리면 D가 쌓이기 시작하고, D에서 출발하는 여러 반응 경로가 **모두 He-4로 합류**한다.

He-4는 핵자당 결합에너지가 가장 높아 **가장 안정**하므로, 도달하면 더 이상 변하지 않는다.

또한, 중성자가 **제한 반응물**이므로 사용 가능한 모든 중성자가 He-4를 만드는 데 쓰인다.

## 질문 2: 화학 반응식 계수 맞추기처럼, 핵반응에서도 무엇이 반드시 보존되는가?

**답:**
핵반응에서는 다음이 보존된다:

1. **질량수(A):** 핵자(양성자+중성자)의 총 개수
2. **전하(Z):** 양성자의 총 개수

예를 들어, $T + D \to {}^4\text{He} + n$에서:
- A: 3 + 2 = 4 + 1 ✓
- Z: 1 + 1 = 2 + 0 ✓

## 질문 3: 여러 반응이 얽힌 네트워크에서 "어느 길이 주요 경로인가"를 어떻게 판단하는가?

**답:**
세 가지 방법으로 판단할 수 있다:

1. **반응률 비교:** 반응률이 높은 경로가 주요 경로
2. **민감도 분석:** 반응률을 바꿨을 때 결과가 크게 변하는 반응이 중요
3. **흐름 추적:** 시뮬레이션에서 어느 경로로 물질이 많이 흐르는지 관찰

BBN에서는 D → T → He-4와 D → He-3 → He-4 두 경로가 주요 경로이며,
모두 He-4로 합류한다.

# 제8장. 오해하기 쉬운 개념

## 오해 1: "He-4가 가장 무거워서 많다"

**정정:** He-4가 많은 이유는 무게가 아니라 **안정성**(결합에너지)과 **네트워크 구조** 때문이다.

## 오해 2: "보존법칙을 만족하면 반드시 많이 일어난다"

**정정:** 보존법칙은 반응의 **가능 여부**만 알려준다.
실제로 많이 일어나는지는 **반응률**에 달려 있다.

## 오해 3: "반응 네트워크가 복잡하면 결과를 예측할 수 없다"

**정정:** 네트워크가 복잡해도 **주요 경로**를 찾으면 결과를 이해할 수 있다.
BBN에서는 D를 거쳐 He-4로 가는 경로가 주요 경로다.

# 부록 A. 수식 보충 (선택 읽기)

## A.1 반응 데이터 구조

코딩에서 반응을 다음과 같이 표현할 수 있다:

```python
# 핵종 정보
SPECIES = {
    'n':   {'A': 1, 'Z': 0},
    'p':   {'A': 1, 'Z': 1},
    'D':   {'A': 2, 'Z': 1},
    'T':   {'A': 3, 'Z': 1},
    'He3': {'A': 3, 'Z': 2},
    'He4': {'A': 4, 'Z': 2},
}

# 반응: p + n → D + γ
reaction_1 = {
    'reactants': {'p': 1, 'n': 1},
    'products': {'D': 1}
}
```

## A.2 보존 검사 함수

```python
def check_conservation(reaction, species):
    """반응의 A, Z 보존을 검사한다."""
    A_left = sum(species[s]['A'] * n for s, n in reaction['reactants'].items())
    A_right = sum(species[s]['A'] * n for s, n in reaction['products'].items())

    Z_left = sum(species[s]['Z'] * n for s, n in reaction['reactants'].items())
    Z_right = sum(species[s]['Z'] * n for s, n in reaction['products'].items())

    return A_left == A_right and Z_left == Z_right
```

- 마지막으로 과학의 미해결 문제를 토이 모델로 탐구한다
